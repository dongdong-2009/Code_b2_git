Some of the files here are generated by the RPC compiler from the provided .x files.
CADICommand.c is the implementation of cadcmd_clnt.c .
CADIEventReceiver.c is the implementation of cadev_svc.c .

In order to compile this project in MSVC, please use the RPCStub project that provides stubbed implementation of rpc calls.

In solaris, the cadi .c & .h files will be automatically generated.
